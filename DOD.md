# Definition of Done (Doc-first)

Für jede Änderung (Commit/PR) gilt minimal:

- [ ] 1 Experiment-Log in `docs/40_experiments/` (Hypothese → Änderung → Messung → Ergebnis)
- [ ] (Optional) 1 ADR in `docs/90_decisions/`, wenn die Entscheidung schwer reversibel ist
- [ ] Tests/Smoke-Run aktualisiert oder ergänzt
- [ ] Metrik-Snapshot ist vorhanden (oder begründet, warum nicht)

Motto: **Doku ist das Produkt. Code ist der Beweis.**
